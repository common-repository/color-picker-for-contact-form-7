== Description ==

<Strong>Color Picker For Contact Form 7</Strong> making Color field to you contact form.

You may create a color picker Field Contact Form 7. Enables adding a color field for Contact Form 7 WordPress Plugin.

<h3>FEATURES:</h3>

<ul>
<li>Required field</li>
<li>Add default preview color and size value</li>
<li>Select color style format(rgba,hex)</li>
<li>Set color popup background color</li>
<li>Set color popup width or height</li>
<li>Set color popup padding</li>
<li>Select color popup opening position</li>
<li>setting id and class of the color picker field (optional)</li>
</ul>
